Please note Peer Group 1 (Insured commercial banks having assets greater than $3 billion) has been broken up into the following three new Peer Groups: Peer Group 1 (Insured commercial banks having assets greater than $100 billion), Peer Group 2 (Insured commercial banks having assets between $10 billion and $100 billion) and Peer Group 3 (Insured commercial banks having assets between $3 billion and $10 billion). Peer groups 2 through 15 have been renamed Peer Groups 4 through 17. Please refer the UBPR User’s Guide and UBPR Peer Group Changes memo dated April 13, 2018 for additional information.

Every national bank, state member bank, and insured nonmember bank is required by its primary federal regulator to file a Reports of Condition and Income (Call Report) as of the close of business on the last day of each calendar quarter (the report date).

The Uniform Bank Performance Report (UBPR) is a multi-page financial analysis of a commercial bank or savings bank that files the Call Report. UBPR data for the most recent quarter is updated daily and for the five most recent consecutive quarters once a week. This schedule allows amendments to Call Report data in prior quarters to be reflected in the UBPR within one week or less.

There are nine UBPR reports available:

  • Executive Summary Report
  • Uniform Bank Performance Report (UBPR)
  • Custom Peer Group Bank Report
  • Peer Group Average Report
  • Peer Group Average Distribution Report
  • State Average Report
  • State Average Distribution Report
  • List of Banks in Peer Group
  • Bank XBRL Document

Search Help

If the search criteria you enter are not sufficiently specific, you may need to refine your search. If the institution you are searching for is not found, consider these:

  • Accuracy – Review the name or city for typos – each must exactly match the way values are stored in the CDR database.
  • Invalid criteria combination – For example, the institution name may be entered correctly, but the city and/or state selected is wrong.
  • Incorrect Unique Identifier – Verify the identifier number entered is correct for the selected type (ID RSSD, FDIC Certificate Number, etc.).
  • Not enough criteria – If too many results are returned, be more specific.

To generate the report:

  1. Select the unique identifier type (such as ID RSSD, FDIC Certificate Number, OCC Charter Number)
  2. Enter all or a portion of the institution’s unique identifier number for the type you selected.
  3. Click the Generate button.

If you enter a portion of the identifier, leading zeros will be added to the number. For example, if you select FDIC Certificate Number as the Unique Identifier, and then enter "345”, two leading zeros will be implied and the search will be conducted using “00345”.

Note that searches can only be performed on the main office level and not on the branch level.

Welcome to the FFIEC Central Data Repository's Public Data Distribution web site. Through this site you can obtain Reports of Condition and Income (Call Reports) and Uniform Bank Performance Reports (UBPRs) for most FDIC-insured institutions. The information available on this site is updated to reflect the most recent data for both prior and current periods. The earliest data provided is from March 31, 2001.

Call Reports are available for individual institutions in Portable Document Format (PDF), Semicolon Delimited Format (SDF), or eXtensible Business Reporting Language (XBRL) format. Bulk data is also available for all reporters in Tab Delimited format or eXtensible Business Reporting Language (XBRL) format.

UBPRs, which are based on Call Report data, are also available online. You can view any of the nine UBPR reports and export the data to a delimited file for further use in Excel or as an eXtensible Business Reporting Language (XBRL) file for use in XBRL enabled tools.

Information on updated products are posted under the News link.

Thrift Financial Report data, filed by thrift institutions through December 31, 2011, is maintained by the Office of the Comptroller of the Currency (OCC). For additional information regarding Thrift Financial Report data, please refer to the information provided by the Office of the Comptroller of the Currency here.

An instance document is a business report defined in the XBRL format. Data within an XBRL instance document can only be interpreted and used with XBRL specific software. It is not intended for use with spreadsheet software such as Excel.
The document contains the same bank financial information displayed within the UBPR system along with other primary business facts that help describe the data - called metadata tagged business facts. To generate the report using the bank’s unique identifier:

  1. Select the report date or a date range.
  2. Select the unique identifier type (such as ID RSSD, FDIC Certificate Number, etc.)
  3. Enter all or a portion of the institution’s unique identifier number for the type you selected.
  4. Click the Generate button.

If you enter a portion of the identifier, leading zeros will be added to the number. For example, if you select FDIC Certificate Number as the Unique Identifier, and then enter "345”, two leading zeros will be implied and the search will be conducted using “00345”.

Note that searches can only be performed on the main office level and not on the branch level.

To generate the report by searching for an institution:

  1. Select the report date.
  2. Enter search values.
  3. Click the Search Banks button to view a list of reports matching the search criteria.

In the Institution Name field, you may enter all or a portion of an institution's legal name, and you may use the percent symbol (%) as a wildcard when searching.

For example, if you enter “Amer%”, all institutions beginning with “Amer” (e.g., American Bank, American Eagle Bank, American Express Centurion, etc.) will be returned. Note that if you enter only the first part of an institution's name, the system will search as if you entered % following that part. In other words, “Amer” and “Amer%” are treated the same way.

If you enter % before the search term (e.g., “%Amer”), all institutions containing “Amer” in its name will be return, such as 1st American State Bank of Minnesota, Advance America Cash Advance Center, Inc., All American Bank, etc.

Note that searches can only be performed on the main office level and not on the branch level.

The Standard report format generates a UBPR using the period selected, one year prior, and three previous year-end dates. For example, if a standard report is chosen for 09/30/2009, the five quarters for the report will be 9/30/2009, 9/30/2008, 12/30/2008, 12/30/2007 and 12/30/2006. The system automatically derives the other four dates from the first date selected.

The Custom report format generates a UBPR using the five dates selected by the user. Each of the five dates is selected using the drop down fields and each date must be filled for the custom report function to work properly.

The Standard report format generates an Executive Summary Report using the period selected, the prior quarter, and one year prior. For example, if a standard report is chosen for 06/30/2020, the quarters for the report will be 06/30/2020, 03/31/2020, and 06/30/2019. The system automatically derives the other two dates from the first date selected.

The Custom report format generates an Executive Summary Report using the three dates selected by the user. Each of the three dates is selected using the drop down fields and each date must be filled for the custom report function to work properly.
